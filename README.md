# マイレシピ (My Recipe App)

レシピ本の写真をアップロードするだけで、AI（Gemini）が料理名、材料、手順を自動で読み取って登録・検索できる、Streamlit製のスマートなマイレシピ管理アプリです。

## 開発のきっかけ・こだわり
毎日の料理の献立決めをラクにしたくて開発しました。
特に、レシピ本やネットのスクショから手入力するのが面倒だったため、Gemini APIの画像解析を使って「写真をアップロードするだけで一瞬で登録できる」ことを目指して作りました。
また、手入力もできるようにしているので、YouTube shortに上がっている画像では登録できないものや、オリジナル料理の登録も大丈夫です！

## 主な機能
- **AI画像解析による自動入力**：レシピ本の写真やスクリーンショットから、料理名・材料・手順を自動抽出します。
- **レシピの保存**: 抽出・編集したレシピをCSVファイル（`recipe.csv`）に簡単保存。
- **複数キーワード検索**: 料理名だけでなく、「ナス 豆腐」のように使いたい食材をスペース区切りでAND検索できます。

## 使用テクノロジー
- **Frontend / Backend**: Streamlit
- **AI (LLM)**: Gemini API (`gemini-3.5-flash`)
- **Language**: Python 3.x
